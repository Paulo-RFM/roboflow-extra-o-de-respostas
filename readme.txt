# Correção Automática de Simulados

## Descrição

Este projeto é destinado à correção automática de simulados. Ele utiliza um conjunto de imagens para avaliação e armazena os resultados em um diretório específico.

## Instalação

Para instalar e configurar o projeto, siga os passos abaixo:

1. Certifique-se de que as pastas `dataset_result` e `respostasFinais` existem no diretório do projeto. Cada uma dessas pastas deve ter suas respectivas subpastas para cada conjunto de dados avaliado.
2. Execute o arquivo `mainMenu.py` para iniciar as predições.

## Uso

Para usar o projeto, execute o arquivo `mainMenu.py` e escolha qual conjunto de imagens você deseja avaliar. Os resultados serão armazenados no diretório `respostasFinais`.

Por favor, note que este projeto foi desenvolvido para uso específico e pode não funcionar corretamente se as pastas e arquivos necessários não estiverem configurados corretamente.
